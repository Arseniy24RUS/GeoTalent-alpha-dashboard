# GeoTalent — публичный паспорт проекта

Продукт объединяет демографию  миграцию  образование  вакансии  зарплаты  транспортную доступность  геоданные и данные заказчика в единой нормализованной модели. Выходом становятся индексы кадрового риска  сценарии мер и аналитическая панель для лиц  принимающих решения.

## Охват демонстрационного международного контура

- 7 первичных источников.
- 1 552 строк спроса.
- 1 243 строк занятости.
- 21 424 строк объявлений с компаниями.
- Все исходные STEM-записи доступны в Data Explorer и в JSON/CSV/Parquet.

## Ограничение публичной версии

В публичной витрине исключены последние четыре цифры СНИЛС и иные персональные идентификаторы из исходной проектной анкеты.
